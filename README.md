# Student Performance Analysis

## Project Overview
Analyzed student dataset to understand performance based on marks, attendance, and study hours using Python.

## Tasks Completed
1. **Task 1: Data Ingestion** - Loaded dataset and displayed first 10 and last 5 rows
2. **Task 2: Data Cleaning & Feature Engineering** - Checked missing values, created Attendance_Group feature
3. **Task 3: Data Analysis & Visualization** - Average marks, Pass/Fail %, Attendance vs Performance, StudyHours vs Marks
4. **Task 4: Conclusion & Report** - Generated summary report with key insights and pass rate

## Tools Used
Python, Pandas, Matplotlib

## Key Insights
- Students with 'High' attendance score higher on average
- Study Hours and Marks have a positive correlation  
- Pass percentage: ~85%
- Overall performance is good
