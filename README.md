# Student Performance Analysis

## Project Overview
Analyzed student dataset to understand performance based on marks, attendance, and study hours.

## Tasks Completed
1. **Task 1: Data Ingestion** - Loaded dataset and displayed first 10 and last 5 rows
2. **Task 2: Data Cleaning** - Removed duplicates, checked missing values
3. **Task 3: Data Analysis** - Average marks, Pass/Fail %, Attendance vs Performance

## Tools Used
Python, Pandas, Matplotlib, Seaborn

## Key Insights
- Students with >75% attendance score higher on average
- Pass percentage: ~85%
